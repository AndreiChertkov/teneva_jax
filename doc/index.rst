Documentation
=============

Python package **teneva_jax** (**ten**\ sor **eva**\ luation with **jax**) provides a very compact implementation of basic operations in the tensor-train (TT) format with jax framework for approximation, optimization and sampling with multidimensional arrays and multivariate functions. The core program code is organized within a functional paradigm and it is very easy to learn and use. Each function has detailed documentation and various usage demos.

-----

Installation instructions are provided in the repository `teneva_jax <https://github.com/AndreiChertkov/teneva_jax>`_. Please, see also our github repository `teneva <https://github.com/AndreiChertkov/teneva>`_, which contains the basic ("numpy") version of the code.
